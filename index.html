Cara Pakai

  1. 1 Download template Excel → isi nama tamu di kolom A (satu nama per baris) → save file
  2. 2 Upload file Excel/CSV atau tambah nama tamu satu per satu secara manual
  3. 3 Klik Kirim ke Berikutnya → WhatsApp otomatis terbuka dengan pesan & link sudah terisi → pilih kontak tamu → Send
  4. 4 Kembali ke halaman ini → klik lagi untuk tamu berikutnya. Status otomatis berubah jadi Terkirim
  5. 5 Data tersimpan di browser — aman refresh halaman. Klik Reset hanya jika ingin mulai dari awal
